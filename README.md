# 🌍 Geospatial 3D Visualizer

An interactive 3D map application built with **Svelte** and **Mapbox GL JS**. This project visualizes urban infrastructure and educational institutions with dynamic environmental effects.

## ✨ Key Features

* **3D Urban Visualization:** Rendered detailed 3D buildings with high-performance extrusion.
* **Dynamic Lighting:** Real-time lighting system that calculates the sun's position based on the current time and location.
* **Infrastructure Highlights:** Specialized layers for schools, kindergartens, and universities with custom interactive tooltips.
* **Global Search:** Integrated Mapbox Geocoder for seamless navigation to any city or address worldwide.
* **Smart Interactions:** One-click navigation buttons for Tbilisi and Rustavi that sync with the search bar.
* **Responsive UI:** Glassmorphism-inspired control panel for a modern look and feel.

## 🛠️ Built With

* [Svelte](https://svelte.dev/) - Frontend Framework
* [Mapbox GL JS](https://www.mapbox.com/mapbox-gljs) - Map Engine
* [Vite](https://vitejs.dev/) - Build Tool

1. Clone the repository:
