# 🌍 Geospatial 3D Visualizer

An interactive 3D map application built with **Svelte** and **Mapbox GL JS**. This project visualizes urban infrastructure and educational institutions with dynamic environmental effects based on real-world time.

## ✨ Key Features

* **3D Urban Visualization:** High-performance 3D building extrusions and realistic depth.
* **Dynamic Lighting:** A real-time system that simulates the sun's position (Dawn, Day, Dusk, Night) based on the viewer's location.
* **Infrastructure Highlights:** Specialized layers for Schools, Kindergartens, and Universities with **interactive glowing tooltips**.
* **Global Search:** Seamless navigation to any address worldwide with a global Day/Night terminator view.
* **Smart Navigation:** One-click shortcuts for Tbilisi and Rustavi that automatically sync with the search bar.
* **Modern UI:** Glassmorphism-inspired control panel and responsive design.

## 🛠️ Built With

* [Svelte](https://svelte.dev/) - Frontend Framework
* [Mapbox GL JS v3](https://www.mapbox.com/mapbox-gljs) - Map Engine
* [Vite](https://vitejs.dev/) - Build Tool

## 🚀 Getting Started

1.  Clone the repository:
    ```bash
    git clone [https://github.com/LaliChidrashvili/Geospatial3DVisualizer.git](https://github.com/LaliChidrashvili/Geospatial3DVisualizer.git)
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Create a `.env` file and add your Mapbox Access Token:
    ```env
    VITE_MAPBOX_TOKEN=your_token_here
    ```
4.  Run the development server:
    ```bash
    npm run dev
    ```
